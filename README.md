# 🌾 AgroCulture - Smart Agriculture Management Platform

AgroCulture is a web-based agriculture management platform designed to empower farmers by providing a digital marketplace where they can directly connect with buyers, manage products, and access agricultural resources. The platform aims to reduce dependency on intermediaries and improve transparency in agricultural trade.

## 📌 Overview

Agriculture remains the backbone of many economies, yet farmers often face challenges in marketing and selling their produce. AgroCulture bridges this gap by offering a centralized platform that enables farmers to showcase products, interact with customers, and manage their agricultural business efficiently.

## ✨ Features

- 👨‍🌾 Farmer Registration and Login
- 🛒 Farmer-to-Buyer Marketplace
- 🌱 Product Listing and Management
- 📦 Inventory Management
- 💰 Product Pricing and Order Handling
- 📝 Agriculture Blogs and Articles
- ⭐ Customer Reviews and Ratings
- 🔍 Product Search and Filtering
- 🔐 Secure Authentication System
- 📊 User-Friendly Dashboard

## 🚀 Installation

### Prerequisites

- PHP 7.x or above
- MySQL
- Apache Server
- XAMPP/WAMP/LAMP

### Steps

1. Clone the repository

```bash
git clone https://github.com/yourusername/AgroCulture.git
```

2. Move the project folder to your server directory.

3. Import the database:

- Open phpMyAdmin
- Create a new database
- Import the provided SQL file

4. Configure database connection:

```php
$servername = "localhost";
$username = "root";
$password = "";
$dbname = "agroculture";
```

5. Start Apache and MySQL.

6. Open the browser:

```text
http://localhost/AgroCulture
```
